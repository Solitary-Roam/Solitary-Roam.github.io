韩国女生离开了
&nbsp;

西班牙小哥也离开了
&nbsp;

在我刚刚要打开的时候
&nbsp;

他们都离开了
&nbsp;

他们静悄悄地来过
&nbsp;

在我身上留下一道划痕
&nbsp;

我不知道
&nbsp;

不知道这划痕
&nbsp;

是否还有意义
&nbsp;

是该留着还是抹去
&nbsp;

可实验室的师姐离开时
&nbsp;

我波澜不惊
&nbsp;

&nbsp;


有人说
&nbsp;

我们终将分别
&nbsp;

当我们相遇的时候
&nbsp;

而这正是我们相遇的意义
&nbsp;

我对这意义有所怀疑
&nbsp;

不过这不重要
&nbsp;

重要的是我知道
&nbsp;

我身上的某些东西
&nbsp;

很难再打开了
